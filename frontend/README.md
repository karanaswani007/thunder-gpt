# Thunder GPT Frontend

Modern, ChatGPT-like chatbot interface for Thunder GPT with Gemini API integration.

## Features

- Modern UI with smooth animations
- Dark/Light theme support
- Chat history management
- Settings for API key configuration
- Responsive design
- Special CSS effects (animations, gradients, hover effects)

## Setup

Simply open `index.html` in a web browser, or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Backend server running on http://localhost:3000
- Valid Gemini API key

## How to Use

1. Open the app in your browser
2. Click Settings (⚙️ icon)
3. Enter your Gemini API key
4. Save and start chatting!
5. Enjoy the smooth animations and beautiful UI

## Browser Support

- Chrome/Chromium 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## File Structure

- `index.html` - Main HTML structure
- `styles.css` - Modern CSS with animations and effects
- `script.js` - JavaScript logic for chat functionality

## Keyboard Shortcuts

- **Enter**: Send message
- **Shift+Enter**: New line in message
- **Esc**: Close settings modal (if open)
